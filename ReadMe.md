This repository contains all PHP lab cycle programs for MCA academics.

### 1. Write a PHP program to show string functions.

### 2. Write a PHP script to check whether the given number is odd or even.

### 3. Write a PHP program to find maximum of three numbers.

### 4. Write a program to show day of the week using switch/case statements.

### 5. Write a PHP program to display Fibonacci series up to a particular count using for loop.

### 6. Write a program to check student grade based on the marks using if-else statement.

>### Conditions:

* If marks are **60% or more**, grade will be **First Division**.
* If marks between **45% to 59%**, grade will be **Second Division**.
* If marks between **33% to 44%**, grade will be **Third Division**.
* If marks are **less than 33%**, student will be **Fail**.
